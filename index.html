Panduan Lengkap Penggunaan

Aplikasi ini dirancang untuk memudahkan Anda menghitung amortisasi portofolio obligasi secara massal menggunakan metode suku bunga efektif. Berikut adalah panduan langkah demi langkah untuk mengisi data dan menggunakan aplikasi ini.

1. Cara Pengisian Input Excel

Agar aplikasi dapat membaca data portofolio Anda dengan benar, baris pertama pada file Excel (baris Header) harus berisi nama-nama kolom berikut. Aplikasi ini mendukung penamaan dalam bahasa Inggris maupun Indonesia.

Kolom Wajib (Harus Diisi):

  • name (atau nama): Nama seri obligasi (contoh: OBLIGASI NEGARA SERI FR 0096).
  • nominal (atau facevalue, par): Nilai pokok/par obligasi dalam bentuk angka penuh tanpa titik/koma (contoh: 20000000000).
  • purchase_date (atau tglbeli, tanggalbeli): Tanggal pembelian obligasi (format: DD/MM/YYYY).
  • maturity_date (atau jatuhtempo, tgljatuhtempo): Tanggal jatuh tempo obligasi (format: DD/MM/YYYY).
  • purchase (atau hargabeli, cost): Harga perolehan/pembelian obligasi saat awal dibeli (contoh: 19751840000).
  • interest (atau bunga, kupon, rate): Tingkat suku bunga kupon tahunan. Bisa ditulis dalam desimal (contoh: 0.07) atau angka utuh/persen (contoh: 7 atau 7%).

Kolom Opsional (Isi jika ada valuasi harga wajar atau penjualan):

  • fair_value (atau nilaiwajar): Nilai wajar (fair value) obligasi pada tanggal cut-off pelaporan. Digunakan untuk menghitung Unrealized Gain/Loss jika obligasi masih aktif.
  • sell (atau hargajual, jual): Harga penjualan obligasi. Diisi jika obligasi telah dilepas sebelum jatuh tempo.
  • sell_date (atau tgljual, tanggaljual): Tanggal pelepasan/penjualan obligasi.
Tips Cepat: Jika Anda tidak ingin membuat dari nol, Anda bisa membuka website tersebut dan mengklik tombol "Unduh Template Excel" untuk mendapatkan format standar yang sudah siap diisi.

2. Cara Penggunaan Website

Aplikasi ini berjalan sepenuhnya di browser Anda, sehingga data finansial yang Anda masukkan tetap aman dan tidak diunggah ke server luar. Berikut adalah langkah-langkah penggunaannya:

  • Siapkan Data Excel: Pastikan file portofolio obligasi Anda sudah diisi sesuai format kolom di atas dan disimpan dalam format .xlsx atau .xls.
  • Unggah File (Upload): Pada bagian Langkah 1 (Data Portofolio Obligasi), Anda bisa:
    • Menyeret (drag) lalu menjatuhkan (drop) file Excel ke dalam kotak abu-abu yang tersedia.
    • Atau, klik kotak tersebut untuk memilih file dari komputer Anda.
    • (Jika berhasil, daftar obligasi Anda akan muncul di bawah kotak).
  • Tentukan Tanggal Cut-off (Valuasi): Pada bagian Langkah 2, klik kolom "Tanggal Valuasi Periode Pelaporan" dan pilih tanggal akhir pelaporan (misalnya 31 Desember 2024). Tanggal ini digunakan sistem untuk menghitung akumulasi amortisasi dan nilai buku (carrying amount) hingga titik tersebut.
  • Proses Perhitungan: Klik tombol hitam besar bertuliskan "Proses Hitung Amortisasi".
  • Baca Hasil (Dashboard & Tabel): Sistem akan langsung memunculkan hasil di bagian bawah:
    • Statistik Utama: Menampilkan ringkasan Total Fair Value, Amortisasi Tahun Berjalan, serta Realized/Unrealized Gain & Loss.
    • Tabel Summary: Ringkasan status dan nilai seluruh obligasi Anda.
    • Jadwal Detail per Obligasi: Di bagian paling bawah, terdapat dropdown menu (pilihan) untuk melihat jadwal buku amortisasi bulan demi bulan untuk masing-masing seri obligasi secara spesifik.
  • Ekspor Hasil ke Excel: Untuk menyimpan kertas kerja perhitungan, klik tombol biru "Unduh Excel" di bagian Ringkasan Eksekutif. File Excel baru akan otomatis terunduh berisi satu sheet summary dan beberapa sheet detail jadwal untuk masing-masing obligasi.
